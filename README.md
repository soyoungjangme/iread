##### Who is user?
> - 책 한 번 읽는 것도 힘든데 그 한 번 읽을 때 많은 걸 얻는 시간이었으면 좋겠는 사용자
> - 남들 다 읽는 건 괜히 읽기 싫은 심리를 가진 사용자
> - 얼마나 팔렸는지가 아닌 얼마나 좋은 책인지를 알고 싶은 사용자
> - 책을 잘 읽고 기록하고 싶은 사용자
> #### 요약
> 더 나은 독서를 위한 템플렛 제공 + 별점없는 리뷰를 통해 나와 fit한 도서 셀렉 환경 제공
